# Dashboard Docs

This folder is the internal home for the `sqldbagent` dashboard surface.

It exists so dashboard-specific behavior does not get mixed into the more
general agent-platform notes.

Use it for:

- Streamlit chat-surface behavior
- thread and persistence behavior
- dashboard-only UX decisions
- schema/prompt/retrieval/query tab notes
- local dashboard debugging and operator workflows

Do not use it for:

- generic LangChain v1 or LangGraph platform guidance
- MCP setup
- general reusable skill-bundle notes
